⚙️ BuildAll: Multi-Chain Skill Portal

Fast reference for agents that need to perform major DeFi/NFT actions through a chat interface.

Supported Chains

Ethereum, Base, Arbitrum, Optimism, Polygon

Major Capabilities

  • ERC-20 (balance, allowance, approve, transfer)
  • ERC-721 (owner, balance, transfer, tokenURI)
  • DEX AMM (Uniswap v3 quote/swap/pool state)
  • Lending (Aave v3 health/market/supply/borrow/repay/withdraw)
  • Stablecoin (Maker basic vault operations on Ethereum)
  • Oracle (Chainlink pair + direct feed reads)
  • NFT Marketplace (OpenSea collection/listing/asset reads)

Quick Start

pnpm install
cp .env.example .env
corepack pnpm typecheck
corepack pnpm test
corepack pnpm build

Then run commands via CLI:

chain-skills -c dex.quote -i '{"protocol":"v3","chain":"base","tokenIn":"0x...","tokenOut":"0x...","amountIn":"1000000","feeTier":500}'

Transaction Safety Pattern

  1. Run with "simulate": true
  2. Confirm addresses and chain
  3. Set slippage guard (amountOutMinimum) for swaps
  4. Send write transaction
